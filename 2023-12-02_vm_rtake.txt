- Standard ロードバランサには Standard パブリック IP アドレスが必要
- VM 作成時に可用性ゾーンを選択できる. この場合, マネージドディスクが必要
- ASP.NET(v4.7) は Win のみ, Ruby(2.6) は Linux のみ, Net Core と PHP(7.3) は両方に対応
- "restartPolocy": "OnFailure" の場合は, コンテナのプロセスが失敗した場合のみ再起動する
- カスタムスクリプトは1回しか実行されないので, 起動のたびに実行したい場合は clout-init を使う
- 障害ドメインが1個のときは更新ドメインは1個に限られる
- 更新ドメインの既定値は1
- リージョナル vCPU は Deallocated でも消費される
- App Service Plan(App Service) はリージョン間で移動しない
- ACI の永続ストレージとして Azure File Storage を使用可能
- 仮想マシンが配置されているサブネットは変更・削除が出来ない
- 仮想マシンを別の vNet に移動させるには新規作成が手っ取り早い
